# 🎲 Ludo Royale

A premium, high-fidelity modern digital adaptation of the classic Ludo strategy board game. Designed with a luxury aesthetic, procedural audio synthesizers, customizable house rules, and an intelligent pathfinding AI.

👉 **[Play Ludo Royale Live Here!](https://farmanullah1.github.io/Ludo/)**

---

## 🎨 Luxury Aesthetic & Visual Design

Ludo Royale is crafted to provide a premium feel, replacing flat styles with a rich jewel-toned layout:
- **Obsidian Dark Mode**: High-contrast, easy-on-the-eyes luxury board frame and background interface.
- **Glassmorphic Interfaces**: Modern overlays, smooth transition modals, and subtle glowing control panels.
- **Responsive Adaptive Board Layout**: The canvas dynamically rescales and repositions to fit mobile viewports, tablet orientations, and wide-screen desktops cleanly without requiring page scrolling.
- **Micro-Animations**: Physics-based rolling dice animations and smooth multi-step token movement paths that match actual tile pathways.

---

## 🎮 Game Modes & Features

- **AI Modes**: Support for 2, 3, or 4 players, allowing any combination of local human players and AI players.
- **Intelligent AI Opponents**: Dynamic pathfinding decision engine with different difficulty behaviors (targeting captures, prioritizing safe zones, and rushing home).
- **Training Mode**: Built-in visual guides, step hints, zero turn timers, and relaxed opponent behavior to help players master the board coordinates.
- **Procedural Synthesizers**: Realistic in-game sounds synthesized on-the-fly using the Web Audio API:
  - Amplitude-modulated rattling for dice rolls.
  - Softer triangle wave pops with transient clicks for token step ticks.
  - Frequency-swept white noise explosions for captures.
  - Pentatonic arpeggios for board entry and victory chimes.
  - Smooth low-pass ambient background chord swells.
- **State Recovery**: Auto-saves active games in real time. If the page is refreshed or closed mid-game, players are greeted with a premium resume prompt to continue right where they left off.

---

## 🚀 Tech Stack

- **Core**: React 19 (Hooks, Context, Reducers)
- **Language**: TypeScript (Strict Mode Validation)
- **Styling**: Tailwind CSS v3
- **Graphics & Logic**: HTML5 Canvas API + RequestAnimationFrame (60fps animation loop)
- **Animation Framework**: Framer Motion
- **Bundler & Dev Server**: Vite 8

## 🛠️ Local Installation & Development

To run Ludo Royale on your local system, follow these steps:

```bash
# 1. Clone the repository
git clone https://github.com/farmanullah1/Ludo.git
cd Ludo

# 2. Navigate to the project root
cd ludo-royale

# 3. Install dependencies
npm install

# 4. Launch local development server (Vite)
npm run dev

# 5. Build production bundle
npm run build
```
