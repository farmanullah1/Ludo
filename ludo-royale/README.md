**Ludo Royale** is a premium, high-fidelity modern adaptation of the classic Ludo strategy board game. Built with a focus on luxury aesthetics, advanced AI, and smooth animations, it brings the tactile feel of a high-end physical board game to your browser.

## ✨ Features

- **Luxury Aesthetic**: Deep obsidian backgrounds, gold gradients, and jewel-toned tokens.
- **Advanced AI**: Three difficulty levels (Easy, Medium, Hard) with intelligent pathfinding.
- **Training Mode**: Learn the game with hints enabled, no timer, and easy opponents.
- **Rich Audio**: Procedural sound effects using Web Audio API and ambient background music.
- **Responsive Design**: Play seamlessly on Desktop, Tablet, or Mobile.
- **High Performance**: HTML5 Canvas rendering for 60fps animations.
- **Customizable**: Multiple player configurations, turn timers, and visual settings.

## 🚀 Tech Stack

- **Framework**: React 18+ (Hooks, Context, Reducer)
- **Language**: TypeScript (Strict Mode)
- **Styling**: Tailwind CSS v3 (JIT)
- **Rendering**: HTML5 Canvas API
- **Animation**: Framer Motion + Canvas RAF
- **Build Tool**: Vite 5

## 🎮 How to Play

1. **Goal**: Move all 4 of your tokens from the base to the center finishing zone.
2. **Starting**: Roll a 6 to move a token out of your home base.
3. **Movement**: Move clockwise around the track. Rolling a 6 gives you an extra turn!
4. **Capturing**: Land on an opponent's token to send it back to their base.
5. **Safe Zones**: Star-marked cells protect your tokens from capture.
6. **Winning**: The first player to finish all tokens wins the game!

## 🛠️ Installation

```bash
# Clone the repository
git clone https://github.com/farmanullah1/Ludo.git

# Navigate to project
cd ludo-royale

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
